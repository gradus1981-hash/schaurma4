Шаурма Гриль - Android project

Что это:
- Готовый Android Studio проект
- Material Design UI
- 2 раздела:
  - Корейская морковка
  - Маринад мяса
- Ввод веса только в граммах

Как собрать APK:
1. Установи Android Studio.
2. Открой папку проекта shaurma_grill_app.
3. Дождись окончания Gradle Sync.
4. Меню Build -> Build APK(s).
5. Готовый APK появится в:
   app/build/outputs/apk/debug/app-debug.apk

Самая короткая сборка:
- Открой проект
- Нажми сверху Run или Build APK(s)

Если Android Studio предложит обновить Gradle или SDK:
- соглашайся на установку Android SDK 34
- после этого проект соберётся
